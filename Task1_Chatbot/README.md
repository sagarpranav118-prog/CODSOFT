# Rule-Based Chatbot

## Overview

This project is a simple Rule-Based Chatbot developed using Python as part of the CodSoft Artificial Intelligence Internship.

The chatbot responds to user inputs using predefined rules implemented through if-else statements.

## Features

* Greeting responses
* Help command
* Name query
* Creator information
* Thank-you responses
* Exit command
* Continuous conversation using loops

## Technologies Used

* Python 3.12

## How to Run

1. Open terminal.
2. Navigate to the project folder.
3. Run:

python chatbot.py

## Sample Commands

* hi
* hello
* how are you
* who are you
* are you interesting
* what is your name
* who made you
* what can you do
* thanks
* help
* bye
