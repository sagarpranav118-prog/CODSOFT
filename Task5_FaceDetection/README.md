# Face Detection System

## Overview

This project is a real-time Face Detection System developed using Python and OpenCV as part of the CodSoft Artificial Intelligence Internship.

The system uses a pre-trained Haar Cascade Classifier to detect human faces through a webcam and draw bounding rectangles around detected faces in real time.

## Features

* Real-time face detection
* Webcam integration
* Multiple face detection support
* Haar Cascade based detection
* Simple and user-friendly interface

## Technologies Used

* Python 3
* OpenCV

## How to Run

1. Open terminal
2. Navigate to the project folder
3. Run:

py face_detection.py

4. Webcam will open
5. Press Q to exit

## Working Principle

The project uses OpenCV's Haar Cascade Classifier for face detection.

Steps:

1. Capture video from webcam
2. Convert frame to grayscale
3. Detect faces using Haar Cascade
4. Draw rectangles around faces
5. Display results in real time
